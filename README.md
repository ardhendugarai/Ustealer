# Ustealer
Ubuntu stealer, steal ubuntu information in local pc (nice with usb key)

## Require
- G++  
`sudo apt-get install g++`

- libsqlite3  
`sudo apt-get install libsqlite3-dev`
## Compilation
Go in `Ustealer/` folder and run makefile  
`make`

## Use
`./Ustealer`
